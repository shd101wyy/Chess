boolean piece.Piece.addToCoordinatesIfValid ( ArrayList< Coordinate >  coords,
int  x,
int  y 
)
inline

Given coordinate (x, y), check whether the piece can move there.

If the piece can move to that coordinate, save (x, y) to coords.

If the piece cannot move anywhere further after reach that coordinate, like there is an another piece there, which blocks the way => return true; otherwise return false

Parameters
coordsthe coordinates array list
xthe x coordinate to check
ythe y coordinate to check
Returns
true if the piece at that coordinate is opponent's piece or player's piece, thus it will block the way.
